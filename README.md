# HireMePortal

HireMePortal is a backend service designed to support a job hiring and resume management platform.  
The system provides APIs for user management, resume analysis, job postings, cart and checkout flows,
and administrative operations.

## Features
- User authentication and role management (Admin / User)
- Resume upload and analysis
- Job description and job visit tracking
- Cart and subscription plan handling
- Order and checkout management
- Admin dashboard APIs
- Notification services

## Tech Stack
- Language: Go (Golang)
- Framework: net/http
- Database: MongoDB
- AI Integration: Resume analysis module
- Tools: Git, GitHub
